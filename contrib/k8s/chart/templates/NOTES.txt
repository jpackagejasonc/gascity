=== Hello-world demo ===

  Sling a task to the coder agent:
    task demo:sling

  Watch the bead (get ID from sling output):
    task demo:watch BEAD_ID=<id>

  Or run directly:
    kubectl exec -n {{ .Release.Namespace }} deploy/{{ include "gas-city.fullname" . }}-controller -- \
      gc sling hello-world/opencode "Create a Python script that prints hello world"

  Open the dashboard (two terminals):
    task demo:port-forward   # terminal 1 — forwards API (9443) and dashboard (7373)
    task demo:dashboard      # terminal 2 — opens http://localhost:7373
